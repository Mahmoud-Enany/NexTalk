# 💬 # 💬 NexTalk | Real-Time Chat Application

A modern real-time chat application built with **ASP.NET Core MVC**, **SignalR**, and **Entity Framework Core**.

NexTalk provides real-time private and group messaging with authentication, friend management, notifications, online presence, typing indicators, message delivery tracking, and a modern responsive interface.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/Mahmoud-Enany/NexTalk.git
```

---

## Restore Packages

```bash
dotnet restore
```

---

## Update Database

```bash
Update-Database
```

or

```bash
dotnet ef database update
```

---

## Run Project

```bash
dotnet run
```
